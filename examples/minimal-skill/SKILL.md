# Minimal Skill

## When To Use

Use this example when you want a tiny valid skill package to try `firety skill lint`.

## Usage

Run `firety skill lint ./examples/minimal-skill`.

## Limitations

This example is intentionally small. It is only meant to show the expected package shape.

## Examples

- Request: "Lint the sample skill."
- Invocation: `firety skill lint ./examples/minimal-skill`
- Result: Review the lint output and use it as a baseline for your own skill package.
